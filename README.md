# learner-model-gql

## Install

This project uses [pnpm workspaces](https://pnpm.io/workspaces).

Install [pnpm](https://pnpm.io/) if not installed already:

```bash
npm i -g pnpm
```

Then execute

```bash
pnpm i
```

And it will install all the required dependencies

## Development

You can execute in the root repo folder:

```bash
pnpm dev
```
