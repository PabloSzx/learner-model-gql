# Database

## PostgreSQL

<br />

<img width="100" height="103" src="/postgres.png" />

This project uses [PostgreSQL](https://www.postgresql.org) as the primary database management system, tested with Postgres v14.1, v13.4, and v13.5.

## Prisma

<br />

<img width="200" src="/prisma.svg" />

This project uses [Prisma](https://www.prisma.io) as Next-generation Node.js and TypeScript [ORM](https://en.wikipedia.org/wiki/Object-relational_mapping).

The database schema is defined in [packages/db/prisma/schema.prisma](https://github.com/PabloSzx/learner-model-gql/blob/main/packages/db/prisma/schema.prisma).
You can read more about how the Prisma schema works in [prisma.io/docs/concepts/components/prisma-schema](https://www.prisma.io/docs/concepts/components/prisma-schema).
