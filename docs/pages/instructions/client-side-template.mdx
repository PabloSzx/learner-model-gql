# Client side template

The client side template for a web application that uses the Learner Model GQL service is available at [`https://github.com/PabloSzx/learner-model-gql-template`](https://github.com/PabloSzx/learner-model-gql-template).

The template uses specific libraries to handle specific needs:

- [Next.js](https://nextjs.org/) - Fully fledged [React](https://reactjs.org/) framework for web applications.
- [TypeScript](https://www.typescriptlang.org/) - Superset of JavaScript with Type-safety in mind. It's encouraged but not mandatory, since TypeScript and JavaScript can be mixed.
- [auth0-react](https://github.com/auth0/auth0-react) - Authentication
- [Chakra UI](https://chakra-ui.com/) - Simple, Modular & Accessible UI Components
- [GraphQL Code Generator](https://www.graphql-code-generator.com/) - Generate code for complete type-safety and easier usage of GraphQL APIs
- [React Query](https://react-query.tanstack.com/) - Hooks for fetching, caching and updating asynchronous data in React
